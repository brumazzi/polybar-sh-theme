# Polybar Configuration

Polybar configuration for i3wm with custom modules and icons!

## Dependences

* gcc >= 14.3 std c21|c++21
* python >= 3.13
* jsoncpp
* FLTK
* curl
* ibus
* jq
* xdotool
* jg_menu
* feh
* alsa
* pactl
* playerctl
* xbacklight
* fontforge
* web3icons

## Fonts
* Roboto
* Awesome 6
* NotoColorEmoji
* Courtier Prime
* HackGen Console
